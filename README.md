#  Healthcare Data Analysis (Python Project)

##  Objective
Analyze healthcare data to understand patterns in treatment cost, patient demographics, and disease distribution.

##  Dataset Features
- Age, Gender, City
- Diagnosis
- Treatment Cost
- Insurance Coverage

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

##  Data Cleaning
- Handled missing values using median and group-wise imputation
- Converted date columns
- Removed inconsistencies

## Key Insights
- Identified most common diseases
- Observed impact of insurance on cost
- Analyzed age vs treatment trends
